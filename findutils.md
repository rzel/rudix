

# findutils #

The GNU Find Utilities are the basic directory searching utilities.

Site: http://www.gnu.org/software/findutils/
License: GPL

Changes from previous release:
**Rebuilt on Xcode 4.3**

Usage:
$ /usr/local/bin/gnufind
