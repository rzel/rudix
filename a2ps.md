# a2ps #

GNU a2ps is an Any to PostScript filter.
Of course it processes plain text files,
but also pretty prints quite a few popular languages.

  * Site: http://www.gnu.org/software/a2ps/
  * License: GPL Version 3

Release Notes:
  * Ported to rudix
